## 打包 TS/TSX

安装 ts-loader 与 typescript

```javascript
module: {
  rules: [
    {
      test: /\.tsx?$/,
      use: "ts-loader",
      exclude: /node_modules/
    }
  ];
}
```

然后创建 tsconfig.json

```json
{
  "compilerOptions": {
    "outDir": "./dist",
    "module": "es6",
    "target": "es5",
    "allowJs": true
  }
}
```
